# Monster Hunter Series Wiki
## 作品概要
本作品は、モンスターハンターシリーズを紹介する非公式Wikiサイトです。
レスポンシブデザインを採用し、PCでは2カラム、スマートフォンでは1カラムで表示されるように設計しました。
また、Pocket TTS（音声合成機能）を利用し、作品紹介やシリーズ紹介を音声で読み上げる機能を実装しています。
掲載している情報は、CAPCOM公式サイトなどの公開情報を参考に作成しており、推測や未確認の情報は掲載していません。

## 主な機能
- モンスターハンターシリーズの紹介
- 作品一覧の表示
- CAPCOM公式サイトへのリンク
- Pocket TTSによる作品紹介の読み上げ
- Pocket TTSによるシリーズ紹介の読み上げ
- スマートフォン・PC対応のレスポンシブデザイン

## 使い方
1. トップページを開きます。
2. 「作品一覧を見る」から作品一覧を確認できます。
3. 「詳細を見る」をクリックすると、CAPCOM公式サイトを新しいタブで開きます。
4. Pocket TTSでは作品を選択し、「作品紹介を読み上げる」を押すと選択した作品を音声で読み上げます。
5. 「シリーズ紹介を読み上げる」を押すと、シリーズ全体の概要を読み上げます。
6. 「読み上げ停止」を押すと、音声を停止できます。
